Лабораторна робота №1

Мета: Ознайомитися на практиці з Python та Django.

Індивідуальне завдання №1: Створіть Django-модель "Категорія" з полями "Назва" та "Опис". Забезпечте можливість призначення статті до конкретної категорії.

Хід вирішення: Для реалізації було створено файл models.py у blog, у якому додано моделі Category і Article. Їм було визначено відповідний зв'язок "один до багатьох" - одній категорії може належати багато статей.

Індивідуальне завдання №2: Створіть Django-модель "Коментар" з полями "Автор", "Текст" та "Дата коментування". Забезпечте можливість додавання коментарів до статей.

Хід вирішення: У файлі models.py було створено модель Comment. Для зв’язку зі статтями використано поле ForeignKey на модель Article. Таким чином реалізовано можливість додавання кількох коментарів до однієї статті та їх подальше отримання.

Індивідуальне завдання №3: Додайте пагінацію для списку статей, щоб користувач міг легко переглядати багато статей на одній сторінці.

Хід вирішення: У файлі views.py було реалізовано функцію article_list, у якій використано клас Paginator для поділу списку статей на сторінки. Поточна сторінка визначається через параметр page у запиті. У шаблоні templates/blog/article_list.html додано елементи керування для переходу між сторінками. Таким чином реалізовано зручну навігацію по великій кількості статей.

Індивідуальне завдання №4: Реалізуйте можливість пошуку статей за ключовими словами. Додайте функціонал пошуку до веб-інтерфейсу.

Хід вирішення: У файлі views.py у функції article_list було додано обробку параметра q з request.GET для отримання пошукового запиту. Використано фільтрацію статей за полями title та content із застосуванням icontains. У шаблоні article_list.html додано HTML-форму для введення пошукового запиту. Також реалізовано перевірку на відсутність результатів пошуку з відповідним повідомленням. Таким чином забезпечено пошук статей через веб-інтерфейс.


Результат: Django-додаток для роботи зі статтями, який включає моделі категорій і коментарів, реалізує зв’язки між ними, підтримує пагінацію списку статей та пошук за ключовими словами через веб-інтерфейс. Додатково було додано простий дизайн інтерфейсу.
