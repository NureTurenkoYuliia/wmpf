Мета: Ознайомитися з базовими можливостями мови Python. Розглянути базові лінійні конструкції, розгалуження, цикли, функції, створення та використання класів.

Індивідуальне завдання №1: Напишіть функцію, яка приймає три параметри (a, b, c) і виводить на екран найменше з них.

Хід вирішення: Порівнюються три числа за допомогою умов if/elif/else, після перевірок виводиться найменше.

Індивідуальне завдання №2: Напишіть функцію, яка приймає рядок та повертає його обернений варіант. Наприклад, "hello" повинно повернути "olleh".

Реалізовано два способи:
- через цикл (побудова нового рядка);
- через зріз [::-1].

Індивідуальне завдання №3: Реалізуйте програму, яка визначає, чи є слово паліндромом (читається однаково з обох боків).
Реалізовано два способи:
- порівняння символів з обох сторін;
- порівняння рядка з його оберненим варіантом.

Індивідуальне завдання №4: Розробіть алгоритм сортування масиву чисел методом швидкого сортування (QuickSort) та виведіть відсортований масив.

Алгоритм QuickSort складається з трьох основних кроків:
- вибір точки опори, елемент з масиву (наприклад, перший елемент або медіана);
- розбиття масиву таким чином, що всі елементи, менші за точку опори, будуть ліворуч від неї, а всі елементи, більші за точку опори, будуть праворуч від неї;
- рекурсивний виклик до двох розбитих підмасивів (рекурсія зупиняється, коли в підмасиві залишається лише один елемент, оскільки один елемент вже відсортовано).

Саме ці кроки було реалізовано для завдання:
- береться pivot як медіана;
- ділиться масив на: < pivot, = pivot, > pivot;
- рекурсивно сортуємо ліву і праву частини;
- об’єднуємо результат.

Результати:
1) Найменше число: 2
2) Обернений рядок: "erun"
3) Перевірка паліндрому: True
4) Відсортований масив: [1, 2, 3, 4, 5, 7, 7, 8, 10]
