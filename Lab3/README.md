# wmpf  

Туренко Юлія
ПЗПІ-23-4

Лабораторна робота №3

Мета: Створення  мобільного застосунку за допомогою Kotlin.


Індивідуальне завдання №1: Створіть базову соціальну мережу. Користувач має мати можливість створити свій профіль, переглядати профілі інших та додавати інших в друзі.

Хід вирішення: Було реалізовано систему користувачів із можливістю створення профілю та вибору активного користувача для подальшої роботи із застосунком. Дані користувачів зберігаються локально за допомогою Room Database. Також реалізовано перегляд списку користувачів, пошук за username та систему додавання друзів.

Індивідуальне завдання №2: Додайте можливість створювати та коментувати пости. Реалізуйте систему дружніх зв'язків між користувачами.

Хід вирішення: Було створено систему постів, де користувач може створювати власні публікації та переглядати пости інших користувачів. Для кожного поста реалізовано коментарі з відображенням username автора коментаря. Також реалізовано систему дружніх зв’язків між користувачами через окрему таблицю Friendship у базі даних.

Індивідуальне завдання №3: Реалізуйте систему особистих повідомлень між користувачами. Додайте систему пошуку користувачів та постів.

Хід вирішення: Було реалізовано приватні чати між друзями з можливістю надсилання та перегляду повідомлень. Повідомлення відображаються у вигляді класичного чату: власні повідомлення справа, повідомлення співрозмовника — зліва. Також реалізовано пошук користувачів за username та пошук постів за текстом контенту.

Індивідуальне завдання №4: Використовуйте компресію (gzip) для передачі даних з сервера на клієнтський інтерфейс для зменшення об'єму даних, які передаються по мережі.

Хід вирішення: Для реалізації gzip-компресії було використано бібліотеку OkHttp. У проєкті створено окремий мережевий клієнт ApiClient, у якому через Interceptor до HTTP-запитів автоматично додається заголовок Accept-Encoding: gzip. Це означає, що клієнт повідомляє серверу про підтримку gzip-стиснення, завдяки чому дані можуть передаватися у стисненому вигляді для зменшення мережевого трафіку та пришвидшення передачі інформації.
